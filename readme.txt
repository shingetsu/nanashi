新月の実装例　NanaSHI 0.0.60.4 by TZR <tzr_jp@hotmail.com>

ネット上の多数の友達、多数の公開ユニットを使用してNanaSHIを
作成しました。この場を借りてお礼を申し上げます。

このアーカイブには新月プロジェクトをDelphi6での実装例が
入っています。TZRの作成物はオープンソースですが、TZR以外
が作成したユニットに関しては、それぞれの使用条件に従って
ください。

この実装例は、新月0.4.X仕様です。(0.0.60.X)　
0.0.61以降は、新月0.5.X仕様の実装例です。
